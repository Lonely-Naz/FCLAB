# Portfolio Videos

Put portfolio videos here.

- `previews/`: short looping preview videos for portfolio cards.
- `full/`: full demo videos if the file is small enough for local hosting.

Recommended format:

- Use `.mp4`.
- Use H.264 encoding.
- Preview duration: `5-15s`.
- Preview size: ideally `3MB - 10MB`.
- Avoid committing very large videos to the repo.

Example names:

- `previews/ai-agent-preview.mp4`
- `previews/dream-motion-preview.mp4`
- `full/ai-agent-demo.mp4`
