# Portfolio Images

Put portfolio images here.

- `covers/`: cover images used by portfolio cards.
- `details/`: larger images used in project detail sections or future detail pages.

Recommended format:

- Use `.webp` when possible.
- Cover width: about `1200px`.
- Detail width: about `1600px`.
- Keep each image around `200KB - 800KB` if possible.

Example names:

- `covers/ai-agent-cover.webp`
- `covers/midjourney-neon-cover.webp`
- `details/midjourney-neon-detail-01.webp`
